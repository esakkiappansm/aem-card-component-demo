# AEM Card Component Demo

## Description
A reusable AEM card component demonstrating dynamic content rendering with editable fields. Ideal for landing pages or feature sections.

## Features
- Dynamic card title, image, and description
- Fully editable template in AEM
- Reusable component across multiple pages

## Tech Stack
Adobe Experience Manager (AEM), HTML, CSS, JavaScript

## How it Works
- Deploy component in AEM
- Edit fields via template editor
